# 🩺 MediExplain AI

MediExplain AI helps users understand complex medical reports using AI.

## Features

- Upload Medical Report PDF
- Simple Patient-Friendly Summary
- Abnormal Findings Detection
- Telugu Explanation
- Questions to Ask Doctor
- Download Summary

## Tech Stack

- Python
- Streamlit
- Gemini AI
- PyMuPDF

## Run Locally

pip install -r requirements.txt

streamlit run app.py
