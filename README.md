# 🩺 MediExplain AI

MediExplain AI is an AI-powered medical report analyzer built for hackathons and healthcare accessibility.

## Features

✅ Analyze PDF Medical Reports

✅ Analyze Medical Report Images

✅ Supports:
- Blood Reports
- CBC Reports
- Thyroid Reports
- Kidney Function Reports
- Liver Function Reports
- MRI Reports
- CT Scan Reports
- X-Ray Reports
- Prescriptions
- Discharge Summaries

✅ Health Risk Dashboard

✅ Abnormal Findings Detection

✅ Patient-Friendly Summary

✅ Test Result Breakdown

✅ Questions To Ask Doctor

✅ Health Recommendations

✅ Multi-language Support
- English
- Telugu
- Hindi

✅ Bring Your Own Gemini API Key (BYOK)

✅ Download Analysis

## Tech Stack

- Streamlit
- Gemini 2.5 Flash
- LangChain
- PyMuPDF
- Pillow

## Future Scope

- Offline Analysis using Ollama
- More Indian Languages
- Hospital Integration
- Medical History Tracking
